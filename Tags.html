Overview

HTML (HyperText Markup Language) is the fundamental language used to structure and present content on the web.It works by using tags and elements to create a hierarchy that browsers can interpret, transforming raw content into the visual webpages we interact with.It forms the backbone of most web pages and applications, organizing text, images, multimedia, and hyperlinks into a readable format.

Tags on HTML

There are so many html tags like:
  • Structural Tags: The tags contains four tags such as < html >, < head >, < body > and < title >.
  • Paragraph Tags: This tag is used for writing any description we would like to express. This tag contains < p > which means paragraph, < br > that means breaking a line into new slot and < hr > means adding horizontal line. Besides < h > tags is used from one to six, in order to resize the paragraph fonts.
  • Text Formatting Tag: This tags contains bold < b >, italic < i > and underline tag < u > respectively.
  • List Tag: This tag is used for enlisting any sentence into dot, numerics, bullets, etc. < ul >, < ol > are the respectively tags for enlisting.
  • Image and Linking Tags: Image and Link tags are the most important tags we use in HTML. < img src > and < a href > defines inserting image and links respectively.
  • Table Tags: Table tags contains three things such as < tr > for table row, < th > for table header and < td > for table data.
